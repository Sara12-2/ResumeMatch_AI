## ResumeMatch AI
IT analyzes how well a resume matches a specific job description using classical NLP techniques — TF-IDF vectorization and cosine similarity for match scoring, spaCy-based Named Entity Recognition for skill/keyword extraction, and rule-based gap analysis to surface exactly what's missing. No LLM calls required for the core scoring engine, making it fast and free to run at scale; an optional Groq-powered layer can turn the raw gap analysis into natural-language improvement suggestions.
